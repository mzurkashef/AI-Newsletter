================================================================================
‚ùì FREQUENTLY ASKED QUESTIONS
================================================================================

Q: How do I get started?
A: Follow PATH 1 or 2 above depending on how much detail you want

Q: Where are my configuration files?
A: config/sources.yaml and config/settings.yaml (edit these)
   .env file (create this with your Telegram info)

Q: How do I add a new newsletter source?
A: Edit config/sources.yaml and add to 'newsletters' section

Q: How do I add a YouTube channel?
A: Edit config/sources.yaml and add to 'youtube_channels' section

Q: How often does it run?
A: Manual: Run whenever you execute the command
   Automatic: Daily at 9:00 AM (configure in Task Scheduler)

Q: Where does it save data?
A: SQLite database at data/newsletter.db (auto-created)

Q: How do I see what it's doing?
A: Check logs/newsletter.log after each run
   Or run with: --log-level DEBUG for more detail

Q: Can I change the delivery time?
A: Yes, edit config/settings.yaml or Task Scheduler properties

Q: Is my Telegram token safe?
A: Yes, stored in .env file which is not in git/public

Q: Can I use email instead?
A: Currently Telegram only. Email support in Phase 2

Q: How do I stop automatic delivery?
A: Delete task: schtasks /delete /tn "AI Newsletter" /f

Q: What if a source goes down?
A: System continues with other sources, logs the error

For more FAQ: See GETTING_STARTED.md ‚Üí "FAQ" section
